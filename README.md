# 📋 Task Manager

A modern and responsive Task Manager built with **HTML**, **CSS**, and **JavaScript**. The application allows users to organize daily tasks efficiently with filtering, sorting, priorities, and persistent local storage.

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit tasks (double click)
- ✅ Mark tasks as completed
- ❌ Delete individual tasks
- 🗑️ Clear all tasks
- 📊 Task counters (Total, Completed, Incomplete)
- 🔍 Filter tasks
  - All
  - Completed
  - Incomplete
- 📌 Task priorities
  - High
  - Medium
  - Low
- 📂 Sort tasks
  - Most Recent
  - Oldest
  - Priority
  - Alphabetical
- 💾 Local Storage support
- 📱 Fully responsive design

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/task-manager.git
```

Open the project folder:

```bash
cd task-manager
```

Run the project by opening the `index.html` file in your browser.
